# Aegis Adapters

> Integration guides for AI coding agents.

---

# Overview

Adapters provide instructions for integrating Aegis with specific AI coding agents. Each adapter includes:
- Agent-specific instructions
- Configuration files
- Best practices

# Usage

1. Choose your agent adapter
2. Copy the instructions file to your project root
3. Configure the agent per the adapter guide
4. Follow ASES phases and quality gates

---

# Creating New Adapters

To create an adapter for a new agent:

1. Create a directory for the agent
2. Add an instructions file (agent-specific name)
3. Add a README with integration guide
4. Add any configuration files
